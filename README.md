# Nodriza Docs

## http://manual.nodriza.io

Nodriza manual statics served from S3 Bucket - docs.nodruza.io

### Install

$ npm run install

### Dev

$npm run dev

### Build production

$ npm run build

### Deploy to S3 Bucket

First copy ./env/env.template to ./env/env and provide the bucket name and AWS Credentials

$ npm run deploy
