# Nodriza Manual
## http://manual.nodriza.io

Nodriza manual statics served from S3 Bucket - manual.nodruza.io

### Install

$ npm run install

### Dev

$npm run dev

### Build for production

$npm run build

### Deploy to S3 Bucket

First copy ./env/env.template to ./env/env and provide the bucket name and AWS Credentials

$ npm run deploy
